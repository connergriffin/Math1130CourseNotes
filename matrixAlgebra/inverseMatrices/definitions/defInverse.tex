\begin{definition}
    Let \(A\) be an \(n \times n\) matrix. The inverse of \(A\), denoted \(A^{-1},\) is the unique matrix such that
    \[AA^{-1} = A^{-1}A = I_{n \times n}.\]
\end{definition}

There are matrices with no inverse.

\begin{definition}
    A matrix is \emph{invertible} if it has an inverse.
\end{definition}

When we are solving a system of linear equations by reducing an augmented matrix \(\begin{bmatrix}
    A & \vert & C
\end{bmatrix}\) we are actually applying a decomposed\footnote{When the product is function composition, we use decomposed where we would normally say factored.} version of the inverse of \(A,\) if \(A\) is invertible. It is somewhat laborious to reduce matrices using the row operations.
