There are some operations on real numbers that either preserve or reverse the relationship between any two real numbers.

\underline{Rules for Inequalities}

These following operations are order \emph{preserving.}

\begin{enumerate}
    \item For any real number, \(c,\) if \(a < b\) then \(a + c < b + c.\) Note the same is true when replace each instance of \(<\) with any one of \(>, \le, \ge.\)
    \item For any positive real number, \(c>0,\) if \(a < b\) then \(ac < bc.\)
    \item For any positive real number, \(c>0,\) if \(0< a < b\) then \(a^c < b^c.\)
\end{enumerate}

The following operations are order \emph{reversing.}

\begin{enumerate}
    \item If \(0<a<b\) then \(\frac{1}{a} > \frac{1}{b}.\)
    \item If \(a<b<0\) then \(\frac{1}{a} > \frac{1}{b}.\)
    
    (Note that for arbitrary \(a<0<b\) we do not know how \(\frac{1}{a}\) compares to \(\frac{1}{b}\))

    \item For any negative real number, \(c<0,\) if \(a<b,\) then \(ac > bc.\)
\end{enumerate}
