\begin{definition}
    A function, \(f,\) is a \emph{rational function} if there are polynomials \(p,q\) such that for all real numbers \(x\) \(f\left(x\right)=\frac{p(x)}{q(x)}.\)
\end{definition}

The domain of the rational function \(\frac{p(x)}{q(x)}\) is the set of all real numbers such that \(q(x)\ne 0.\)

    Which of the following are rationial functions?

        \choice[correct]{\(f(x) = 2\)}
        \choice[correct]{\(g(x) = -1+x\)}
        \choice[correct]{\(h(t) = t^2+2t-1\)}
        \choice[correct]{\(r(x) = \frac{1}{1-x}\)}
        \choice[correct]{\(r(t) = \frac{4}{3}t^3-1\)}
        \choice{\(f(x) = \sqrt{x^2+1}\)}
        \choice[correct]{\(g(t) = \frac{t+1}{t^2+1}\)}
    \end{selectAll}
\end{question}
