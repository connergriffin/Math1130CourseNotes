\begin{example}
    Convert the following exponential equations to logarithmic equations.
    \begin{enumerate}
        \item \(6^2=36\)
        \item \(2^4=16\)
        \item \(49^{1/2}=7\)
        \item \(10^3=1000\)
    \end{enumerate}

    \begin{solution}
    \begin{enumerate}
        \item \(\log_6(36)=2\)
        \item \(\log_2(16)=4\)
        \item \(\log_{49}(7) = \frac{1}{2}\)
        \item \(\log(1000) =3\)
    \end{enumerate}
    \end{solution}
\end{example}

This same conversion applies to approximations.
\[\textrm{Since,} \ e^{3}\approx 20.08 \ \textrm{we have} \ \ln(20.08) \approx 3.\]
